# Founders Notes — agent guide

Personal knowledge vault: Founders podcast transcripts, study notes, and X posts.

## Start here

1. **Episode index:** [`catalog/episodes.jsonl`](catalog/episodes.jsonl) — ids, URLs, `transcript_status`, paths
2. **Gaps / coverage:** [`catalog/gaps.md`](catalog/gaps.md) — auto-generated by `ingestion/verify.py`
3. **Import issues:** [`catalog/import-review.md`](catalog/import-review.md), [`catalog/post-mapping-review.jsonl`](catalog/post-mapping-review.jsonl)
4. **X raw cache:** [`import/x-posts-raw.csv`](import/x-posts-raw.csv) (gitignored) — sync first, organize second. Replies to other users are excluded from attribution.

## One episode

For `ep-NNN`, use the same folder basename under:

- `content/transcripts/{folder}/{folder}.md`
- `content/notes/{folder}/notes.md` (raw datapoints)
- `content/notes/{folder}/expanded.md` (optional, after datapoint workflow)
- `content/posts/{folder}/post.md`

Resolve `{folder}` from catalog `id` + `slug` (see [`docs/episode-id-rules.md`](docs/episode-id-rules.md)).

## Cross-episode search

```bash
cd ingestion
python search.py "rockefeller"
python build_chunks.py   # refresh catalog/chunks.jsonl first
```

Or ripgrep: `content/transcripts/`, `content/notes/`, `content/posts/`, `content/posts/_corpus/all-posts.md`

## Datapoint expansion

User workflow: timestamp bullets in `notes.md` → full quotes + takeaways. See [`docs/datapoint-workflow.md`](docs/datapoint-workflow.md).

```bash
python expand_datapoints.py --id ep-200
```

## Ingestion (minimal)

| Task | Command |
|------|---------|
| New episodes | `python sync_new.py --repair-urls --apply` |
| Apple Notes | `python import_notes.py -i ../import/apple-notes.txt` |
| X cache sync | `python sync_x_cache.py` (incremental) or `--full` (backfill) |
| X organize | `python organize_posts_from_csv.py` (reads CSV only) |
| Manual post | `python assign_post_manual.py --episode N --x-post-id ID --published-at YYYY-MM-DD --body-file path` |
| Dedupe CSV | `python dedupe_x_csv.py` (after overlapping `--full` syncs) |
| Chunk index | `python build_chunks.py` |
| Verify | `python verify.py` |

Do not add embeddings unless grep + chunk index fail for the user's post corpus. See [`docs/retrieval.md`](docs/retrieval.md).
